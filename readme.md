## Getting Started

Now the application only works with Docker. 

### Prerequisites

Download <a href="https://www.docker.com/">Docker</a>.

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:CodecoolGlobal/el-proyecte-grande-sprint-1-java-csakafeda.git
   ```
2. Run Docker.
3. Run the application on Docker.
   ```sh
   docker compose up -d
   ```
